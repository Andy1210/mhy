---
description: >-
  There are some optional flags that are coming with the CLI tool. All flags are
  being prefixed with `mhy` to prevent conflicts with other processes.
---

# options

## --mhy-help

```bash
# show usage details
mhy --mhy-help
```

## --mhy-debug

```bash
# show debug messages
mhy --mhy-debug
```

## --mhy-verbose

```bash
# show more info than usual
mhy --mhy-verbose
```
